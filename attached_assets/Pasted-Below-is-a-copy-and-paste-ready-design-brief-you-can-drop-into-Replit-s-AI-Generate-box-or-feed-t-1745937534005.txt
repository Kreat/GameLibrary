1 Brand & Visual Identity

Element	Spec
Nameplate	GameHub (semibold, tracking-wide)
Primary palette	#FFD45C (“Meeple Gold”), #2B2D42 (Slate Night), #EF476F (Boardwalk Red), #06D6A0 (Mint Token)
Accent palette	#FFFFFF paper-white, #F4F4F5 light gray for surfaces
Typography	Heading: font-display = Poppins, 700  Body: font-sans = Inter, 400/500
Shadow & radius	Card radius rounded-lg; elevation via Tailwind shadow-md hover:shadow-lg

	Rationale: The saturated gold + red evoke classic board-game pieces; deep slate anchors contrast. Simple sans fonts keep the site feeling modern rather than “fantasy.” These colors stay WCAG-AA at 4.5:1 against white backgrounds.

2 Information Architecture (IA)
	1.	Landing (/) – marketing hero + “Find a Session” CTA
	2.	Dashboard (/app)
	•	Left sidebar → filters & nav
	•	Main feed → Session Cards (see §3)
	3.	Session Wizard (/wizard) – multi-step form (time → people → place → review)
	4.	Leaderboard (/leaderboard) – top hosts / players (pulls Algolia index)
	5.	Profile (/u/[uid]) – avatar, reputation, upcoming RSVP list
	6.	Admin (/admin) – CRUD tables if role === "admin"

3 Core React Components

Component	Purpose	Key props
<SessionCard>	Shows one upcoming game	title, system, dateTime, host, seatsFilled/total, location, onRSVP()
<AvailabilityChip>	Small colored pill for each player’s time window	start, end, clashLevel
<WizardStep>	Reusable step shell for the Session Wizard	stepNumber, title, children, onNext()
<ReputationBadge>	Displays numeric rep + tooltip	score
<RoomMap>	Static Mapbox image + pin overlay	roomId

All components fully typed in TSX; style exclusively with Tailwind utility classes—no external CSS files.
